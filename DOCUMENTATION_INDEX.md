| Issue | Solution |
|-------|----------|
| Port 3000 in use | `npm run dev -- -p 3001` |
| Styles not updating | `rm -rf .next && npm run dev` |
| Audio not playing | Check `public/sounds/` directory |
| Dependencies error | `npm cache clean --force` then reinstall |
